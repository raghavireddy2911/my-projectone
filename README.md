# To-Do List Application

## Overview

This is a simple command-line To-Do List application built using Python. It helps users manage their daily tasks by allowing them to add, view, and delete tasks. Tasks are stored in a text file so they remain available even after the program is closed.

## Features

* Add new tasks
* View all tasks
* Delete existing tasks
* Save tasks permanently using file handling
* Load saved tasks automatically when the program starts

## Technologies Used

* Python
* File Handling

## How to Run

1. Install Python on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the following command:

python todo.py

## Project Structure

* `main.py` - Main application code
* `tasks.txt` - Stores saved tasks
